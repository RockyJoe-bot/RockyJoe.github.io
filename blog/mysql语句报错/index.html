Mysql语句报错

Aug 14 2020

在程序运行的时候Mysql语法报错
check the manual that corresponds to your MySQL server version for the right syntax to use near ‘-5，5’
看似是语法错误，首先检查Mysql分页语句语法并无错
其次检查分页代码，经过debug之后发现model的size=0
之前用flyway做数据迁移的时候字段并没有迁移过去，数据库内没有字段，所以首页读不出来关联数据库的数据模型。报的语法错误也是因为找不到相应的数据库字段，于是将字段补上就能够运行了。
